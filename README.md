# TCP to HTTP in Go

API for Pulse App 

## Prerequisites

### Required

- [Go](https://go.dev/) 1.24.2+

### Optional

- [Just](https://just.systems/) to run commands

## Getting started

Obviously, you also need to clone this repository:

```sh
git clone git@github.com:kensonjohnson/tcp-to-http-in-go.git
```

Fetch the dependencies:

`just tidy` or `go mod tidy`

Then, you can run the server using the following command:

`just run` or `go run .`
